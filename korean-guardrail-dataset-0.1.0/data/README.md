# 📂 data

```text
data/
├─ processed/
├─ translated/
├─ raw/
└─ src/
```

## /processed
> 원본 데이터셋을 검증 및 테스트용으로 가공하여 JSONL 형식으로 저장한 데이터셋 모음입니다.

## /translated
> 원본 데이터셋을 번역하여 JSONL 형식으로 저장한 데이터셋 모음입니다.

## /raw
> 각 원본 데이터셋 모음입니다. (원본 데이터 중 용량이 큰 데이터는 직접 **source link** 를 통해서 다운로드 해주세요)

## /src
> 원본 데이터 **data/raw**를 AI Agent 서비스 평가 및 검증용 데이터셋을 변역/생성/검증하는 스크립트 모음입니다. (원하는 형식으로 변경하여 활용하세요)
